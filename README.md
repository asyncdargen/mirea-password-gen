# Простой генератор паролей

Коротко:
Приложение на React. Генерирует 5 паролей и позволяет копировать их по отдельности или все сразу.

Структура проекта:
- package.json
- index.html
- src/
  - main.jsx
  - App.jsx
  - index.css
  - utils/
    - password.js
- .gitignore
- README.md
